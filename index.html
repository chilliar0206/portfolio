Cinemartyrs.com

technologies used: HTML, CSS, Bootstrap, PHP, SQL, GIMP Photoshop

Cinemartyrs.com is a PHP based movie blog. When I was designing it, I wanted to give it a unique color scheme that took influence from classic Hollywood and modern movie theaters. The website was built with Bootstrap to make sure it was responsive to mobile screens. I also made the decision to create a CMS for the site to get a better understanding of how content management systems work and how they can be manipulated. The website is still a work in progress, with updates planned, but it is currently being hosted and content is being added to it regularly.

My Portfolio

technologies used: HTML, CSS, Bootstrap, Vanilla JavaScript

When I set out to make this portfolio, I wanted to make a simple, single page application where nearly everything was one click away. I also wanted to avoid the long web page design that many portfolio pages share, where it seems like the pages scroll for over 2000px. By using custom Vanilla JavaScript functions and CSS transitions, I was able to keep everything contained to one part of the page, keeping the page short, simple, and easy to navigate. The simple color scheme was chosen to reflect the color scheme of the text editor I use.

Budget Calculator

technologies used: JavaScript

This is a monthly budget calculator web application that I made while taking a JavaScript course. The HTML and CSS were provided by the instructor. The application allows users to input expected incomes and expenses for one month and the application will produce a budget for that month.
